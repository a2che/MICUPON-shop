Есть несколько моментов, которые по моему мнению стоит изменить:

1) размеры ширины для всех экранов, ибо есть определенные стандарты для экранов разных
размеров ( например ни у кого нет телефонов 600px на 700px, есть по 320, 375, 425,
ориентироваться лучше на эти цифры, соответственно ширина некоторых элементов может
отличаться. Но на ширине устройства в 600px всё будет по макету.)

Предложение: <376px - для мобильной версии, <768px - для планшетов и маленьких экранов,
<1440px - для больших экранов. ( Вам изменять ничего не нужно, просто, ориентируясь
именно на эти размеры, дизайн будет хорошо смотреться на любом экране)

2) На экране регистрации иконка ВК отличается от Одноклассников и ФБ, желательно взять
иконку с синей заливкой, а не синими краями ( так будет смотреться более элегантно )
